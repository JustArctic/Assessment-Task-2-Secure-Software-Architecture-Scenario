# Perfect Pizzas - Assessment Task 2 Secure Software Architecture Scenario

A safer and more fault tolerant version of the Flask Cooking Blog coded by group 1 in Programming For the Web

## Features 
- Protection against XSS Attacks, CSRF Attacks, Brute Force Attacks and Invalid Redirects
- Two Factor Authentication
- HTTPS and SSL certificates
- Multiple methods of encryption for passwords and emails
- Graceful error handling and session management
- And more!

## Installation Guide
### Requirements
- Python (version 3.x or higher) and PIP (version 25.x or higher)
- MKcert (version 1.4.x or higher)

### Step-by-Step Installation
1. Click the green "Code" button on GitHub page and click "Download ZIP"
2. Navigate to **FlaskBlog-group-task-main.zip** and open it by **Extracting** or third-party software such as [WinRAR](https://www.win-rar.com/download.html?&L=0)
3. Drag out the **Flask_Blog** from inside the ZIP file or from the folder created by **extracting** (if extraction method was used) to any suitable location of your choosing. You can also Right-Click the ZIP file and press Extract All to create a folder.
5. Double-click **run.bat file** to begin hosting website locally
6. **Ctrl + Left-Click** website URL in cmd prompt to access website
